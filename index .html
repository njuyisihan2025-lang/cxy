需求测算模型
输入区域峰谷价差、负荷结构、新能源装机比例、政策强度，输出各区域储能需求空间和分场景需求占比。
IRR 测算模型
输入系统成本、峰谷价差、利用小时数、融资成本、辅助服务与容量补偿，输出 IRR、回收期和盈亏平衡点。

这份 demo 怎么发布到 GitHub

你不用学代码，只需要把这个文件上传到仓库并打开 GitHub Pages

  1. 把这个文件命名成 index.html
  2. 上传到你的 GitHub 仓库根目录
  3. 进入 Settings → Pages
  4. 选择 Deploy from a branch
  5. 分支选 main，目录选 / (root)
  6. 保存后，GitHub 会给你一个公开网址
